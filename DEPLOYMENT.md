# 🚀 LocalEyes Deployment Guide

This guide will help you deploy your LocalEyes application to the cloud so multiple users can access it from different devices.

## 📋 Prerequisites

- GitHub account
- Railway account (free tier)
- Vercel account (free tier)

## 🎯 Deployment Steps

### Step 1: Push to GitHub

1. Create a new repository on GitHub
2. Push your code:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/yourusername/localeyes.git
   git push -u origin main
   ```

### Step 2: Deploy Backend to Railway

1. Go to [Railway.app](https://railway.app)
2. Sign up with GitHub
3. Click "New Project" → "Deploy from GitHub repo"
4. Select your repository
5. Choose the `server` folder as the root directory
6. Add PostgreSQL database:
   - Click "New" → "Database" → "PostgreSQL"
7. Set environment variables:
   - `DATABASE_URL` (auto-generated by Railway)
   - `NODE_ENV=production`
   - `FRONTEND_URL=https://your-frontend-url.vercel.app` (update after frontend deployment)

### Step 3: Deploy Frontend to Vercel

1. Go to [Vercel.com](https://vercel.com)
2. Sign up with GitHub
3. Click "New Project" → "Import Git Repository"
4. Select your repository
5. Set build settings:
   - Framework Preset: Vite
   - Root Directory: `./` (root)
   - Build Command: `npm run build`
   - Output Directory: `dist`
6. Add environment variable:
   - `VITE_API_URL` = Your Railway backend URL (e.g., `https://your-app.railway.app`)

### Step 4: Update CORS Settings

1. Go back to Railway dashboard
2. Update the `FRONTEND_URL` environment variable with your Vercel URL
3. Redeploy the backend

## 🔗 Your Live URLs

- **Frontend**: `https://your-app.vercel.app`
- **Backend**: `https://your-app.railway.app`
- **API Health**: `https://your-app.railway.app/api/health`

## 🧪 Testing Multi-Device Access

1. Open the frontend URL on your phone
2. Create a new account
3. Report an issue
4. Open the same URL on another device
5. Login and verify the issue appears

## 🎉 Success!

Your LocalEyes application is now live and accessible from any device worldwide!

## 🔧 Troubleshooting

### Backend Issues
- Check Railway logs for errors
- Verify environment variables are set
- Ensure PostgreSQL database is connected

### Frontend Issues
- Check Vercel build logs
- Verify `VITE_API_URL` is correct
- Check browser console for CORS errors

### CORS Errors
- Update `FRONTEND_URL` in Railway environment variables
- Redeploy backend after changes
